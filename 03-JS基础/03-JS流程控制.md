# 流程控制

## switch

> **switch底层用的是 === 比较。**

参考链接：http://www.cnblogs.com/codetker/p/4680996.html

```javascript
switch ("111"){
    case 111:
        alert(111);
        break;
    case 222:
        alert(222);
        break;
    case 222:
        alert(333);
        break;
    default :
        alert("都不是！");
}
```
